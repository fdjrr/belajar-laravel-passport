# Belajar Laravel Passport

# Installation

```bash
$ git clone https://github.com/fdjrr/belajar-laravel-passport
$ cd belajar-laravel-passport
$ composer update
$ php artisan migrate
```
# Create Client using CLI

```bash
$ php artisan passport:client
```
